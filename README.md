# 学生语音朗读评分系统

一个专为微信群里学生辅助学习设计的网页应用，帮助老师纠正学生发音并批改听写作业。

## 主要功能

### 1. 课文朗读评分
- **录音功能**：学生可直接在网页上录制朗读
- **文件上传**：支持上传音频(MP3/WAV/M4A)和视频(MP4/WebM)文件（最大50MB）
- **拖拽上传**：可拖拽文件到上传区域，支持批量上传
- **发音分析**：自动分析发音准确性、语调流畅度和节奏掌握
- **问题识别**：自动识别有问题的单词并给出标准发音建议
- **评分系统**：提供综合评分和详细分数分析
- **内置播放器**：上传的音频文件可直接在网页内播放

### 2. 听写批改
- **听写批改**：输入听写原文和学生答案，自动批改
- **错误纠正**：自动识别错词并显示正确拼写
- **分数统计**：计算听写得分和正确率
- **批改建议**：针对错误类型提供针对性的练习建议

### 3. 班级管理（开发中）
- 学生成绩统计
- 作业布置与收集
- 学习进度跟踪
- 微信群对接

## 使用场景

- 英语老师：纠正学生发音，批改听写

1. **课前预习**：学生提前练习新课生词发音
2. **课堂练习**：课堂上进行实时发音练习
3. **课后作业**：布置朗读作业，系统自动评分

## 技术特点

### 前端技术
- 纯HTML/CSS/JavaScript实现，无需后端服务器
- 响应式设计，支持手机、平板、电脑
- 使用浏览器内置录音API
- 本地存储，保护用户隐私

### 发音分析
- 基于常见发音问题的智能分析
- 包含常用英语单词的音标库
- 根据单词难度动态调整评分标准
- 针对性的练习建议生成

## 快速开始

### 方法一：直接使用（最简单）
1. 将 `index.html` 文件分享到微信群
2. 学生点击链接即可开始使用

### 方法二：本地运行
1. 下载所有文件到本地文件夹
2. 用浏览器打开 `index.html`
3. 允许浏览器访问麦克风权限

### 方法三：部署到服务器
将整个文件夹上传到任何静态网页托管服务：
- GitHub Pages
- Netlify
- Vercel
- 腾讯云静态网站托管

## 使用指导

### 对于老师
1. **布置朗读作业**：
   - 在"朗读评分"模块更换练习内容
   - 将页面链接分享到微信群
   - 让学生录音后分析发音

2. **批改听写作业**：
   - 在"听写批改"模块输入听写原文
   - 输入或上传学生答案
   - 点击批改，生成批改结果

3. **查看学生进步**：
   - 记录学生每次练习的分数
   - 根据问题单词制定针对性教学计划
   - 将优秀的朗读录音发到群里表扬

### 对于学生
1. **练习朗读**：
   - 点击"开始录音"录制朗读
   - 完成后点击"分析发音"
   - 查看问题单词和改进建议
   - 可重复练习直到满意

2. **提交听写**：
   - 将手写听写拍照上传
   - 或直接输入答案文本
   - 查看批改结果和错误纠正

3. **上传录音/视频作业**：
   - 支持拖拽上传，操作简单
   - 支持多文件批量上传（最多10个）
   - 上传后可直接播放检查内容
   - 系统分析发音质量并提供改进建议

### 📁 音频/视频文件上传功能详解

#### 支持格式：
- **音频格式**：MP3 (.mp3)、WAV (.wav)、M4A (.m4a)、WebM Audio (.webm)
- **视频格式**：MP4 (.mp4)、WebM Video (.webm)
- **最大文件**：50MB
- **拖拽上传**：直接拖拽文件到指定区域
- **批量选择**：按住Ctrl/Cmd键可选择多个文件

#### 使用流程：
1. 点击"选择文件"按钮或拖拽文件到上传区域
2. 查看文件列表，确认上传内容
3. 点击"分析音频"按钮开始处理
4. 查看评分结果和详细分析
5. 可在网页内直接播放上传的音频文件

#### 教学应用价值：
1. **学生作业提交**：用手机录音上传，方便老师批改
2. **老师示范**：录制标准发音示范，学生反复聆听
3. **对话练习**：上传英语对话录音，分析双方发音
4. **考试存档**：重要口语考试录音保存，便于复习评估

## 核心功能详解

### 发音评分算法
1. **基础评分**：基于文本难度计算理想分数
2. **难度调整**：根据难词比例调整评分
3. **随机波动**：模拟用户实际表现差异
4. **分数范围**：确保分数在30-98合理范围内

### 单词问题识别
1. **优先级识别**：优先识别难词问题
2. **发音建议**：提供具体的发音改进建议
3. **音标标注**：显示标准音标供参考
4. **互动学习**：点击单词可听标准发音建议

### 批改系统
1. **自动比对**：将学生答案与原文逐词比对
2. **错误识别**：高亮显示错误单词
3. **正确统计**：统计正确率生成分数
4. **建议生成**：根据错误类型提供针对性建议

## 隐私保护

### 数据安全
- 所有录音仅保存在用户本地
- 不收集任何个人身份信息
- 不发送录音到任何服务器
- 所有分析在浏览器内完成

### 使用建议
- 建议在安全网络环境下使用
- 使用后清除浏览器缓存
- 如有敏感内容，注意保护隐私

## 技术支持

### 常见问题
1. **麦克风权限被拒绝**
   - 检查浏览器设置
   - 刷新页面重新授权

2. **录音无法播放**
   - 检查浏览器是否支持录音功能
   - 尝试使用Chrome或Edge浏览器

3. **音标显示异常**
   - 确保使用现代浏览器
